# Media Catalog 🎬📚🎮

**Media Catalog** — это полнофункциональная личная картотека для каталогизации фильмов, книг и видеоигр. Проект представляет собой веб-приложение, которое помогает пользователям систематизировать свою медиатеку, отслеживать прогресс просмотра/прочтения, делиться впечатлениями и открывать для себя новый контент на основе персональных предпочтений.

## 🌟 Ключевые возможности

- **Управление медиатекой**: добавление, редактирование и удаление фильмов, книг и игр
- **Система оценок и отзывов**: возможность оценивать произведения по 10-балльной шкале и оставлять развернутые рецензии
- **Статусы просмотра**: отметки "Просмотрено/Прочитано", "В процессе", "В планах"
- **Персональные коллекции**: создание тематических подборок произведений
- **Импорт данных**: пакетное добавление произведений через CSV-файлы
- **Умные рекомендации**: персонализированные подборки на основе предпочитаемых жанров
- **Фильтрация и поиск**: удобный поиск по названию, жанру, году выпуска и типу произведения
- **Статистика**: наглядная аналитика по вашей медиатеке
